# Catchuco Command Line Sound

> Tem um cathuco bem no mei! 

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g catchuco

## Running
Run command:

    cathuco

Or import in NodeJS script:

    import daledale from 'cathuco';

    cathuco();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx cathuco
